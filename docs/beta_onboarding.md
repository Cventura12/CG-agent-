# Beta Contractor Onboarding

Arbor Agent turns quick voice notes into a draft quote, a customer-ready PDF, and a live queue of follow-up actions.

## What The App Does

1. Tap `New Quote`.
2. Hold the mic button and speak field notes like roof size, address, and job type.
3. Release to send.
4. Review the generated quote.
5. Tap `Send PDF` to create the customer-facing PDF and share it by text or email.
6. Return to `Queue` to approve, edit, or discard follow-up drafts.

## How To Use Voice Capture

- Use the app on your phone, not your laptop.
- Hold the orange mic button while speaking.
- Speak in this order when possible:
  - customer name
  - project address
  - roof type or repair type
  - roof size / squares
  - special conditions (steep, decking damage, insurance claim, etc.)
- Release the button when finished. The app will send the transcript automatically.
- If voice capture fails, type the same notes into the text box and tap `Send Notes`.

## If A Quote Looks Wrong

1. Do not send it immediately.
2. Check the address, scope, and price summary first.
3. If the scope is wrong:
   - rerun the quote with clearer field notes
   - include the missing detail directly in the transcript
4. If the pricing is off:
   - use the Chattanooga baseline only as a temporary default
   - tell us which line item is wrong so your contractor profile can be updated
5. If the PDF looks correct but needs wording changes:
   - update the text before sending
   - then tap `Send PDF`

## What To Report During Beta

- Quotes that miss obvious job details
- Wrong material assumptions
- Voice capture misses or cut-off transcripts
- PDF layout issues on your phone
- Anything that makes you stop and think instead of moving fast

## Beta Use Standard

Use the app on a real phone and a normal mobile network. The goal is to verify the full path:

`voice -> quote -> PDF -> send`
